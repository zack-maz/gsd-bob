## overview
make open-gsd natively compatible with IBM Bob
open-gsd: https://github.com/open-gsd/gsd-core
bob docs: https://bob.ibm.com/

## instructions
- review open-gsd and how it works
- identify all slash commands, subagents, workflows, etc that can we transfer over to IBM Bob
- identify IBM Bob architecture, how it works with slash commands, modes, etc
- translate open-gsd to work natively in IBm Bob architecture with claude code cli, gemini, etc
- create a one-line install that sets up GSD in a user's environment
    - user can select between update, clean install, etc
    - user can select between local and global scope
- somehow extend/contribute to open-gsd/gsd-core repo (as its open-source)
