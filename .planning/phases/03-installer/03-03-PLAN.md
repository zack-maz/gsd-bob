<objective>
Build the staging engine: `stage.cjs` orchestrates the three structural pieces every install stages (gsd mode merge ‚Üí custom_modes.yaml, vendored `gsd-core/` payload copy, SUPPORT-ROSTER.md regeneration) plus a roster-agnostic convertible-artifact loop that tolerates the v1 reality that NO `commands/gsd/` source is vendored yet (RESEARCH Pitfall 1). Output: `src/installer/stage.cjs`, `src/installer/config-merge.cjs`, two house-style test files using a scratch tmpdir `.bob/` target.
</objective>
