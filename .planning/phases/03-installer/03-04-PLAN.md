## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI invocation → filesystem | the entry mutates the user's filesystem (write/overwrite/delete) under the resolved scope target |
| manifest on disk → uninstall | the manifest read at uninstall decides what is deleted vs un-merged |
