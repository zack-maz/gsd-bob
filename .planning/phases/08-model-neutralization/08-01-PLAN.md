<objective>
Add a durable model-neutralization pass and a zero-literal invariant so every Bob-native artifact gsd-bob EMITS THROUGH ITS CONVERTERS (`.bob/commands/gsd-*.md` + `.bob/skills/gsd-*/SKILL.md`) carries zero model-routing references — Bob owns model routing. Purpose: Today's 10-command emission is already effectively neutral (the vendored converters strip `model:`/`effort:` frontmatter by whitelist rebuild — F-02; bodies reference workflows by path — F-03). This phase is INSURANCE for Phase 9, not a cleanup of today's output: a composable post-pass that rewrites any surviving tier prose, plus a loud invariant that guards it against regression, both driven by ONE shared regex so they cannot drift.

Output: `neutralizeModelReferences` + `scanModelLiterals` + shared regex/replacement constants in `src/bob-adapter.cjs`; the post-pass wired into `stage.cjs`; a new `test/model-neutrality.test.cjs` invariant; a corrected `test/acceptance-coverage.test.cjs`; and an insert-only AC-27 acceptance step.

Scope is LOCKED to the emitted converted set (D-01, research-affirmed). The raw-copied `.bob/gsd-core/**` payload is OUT of scope — its `bin/` is executable model-resolution runtime and its `model_profile`/`resolve_model_ids` are GSD config vocabulary, not routing directives. NEVER edit the vendored converter `gsd-core/bin/lib/runtime-artifact-conversion.cjs` (D-02, preserves Phase 7's verbatim-payload invariant and the upstream move-not-rewrite story).
